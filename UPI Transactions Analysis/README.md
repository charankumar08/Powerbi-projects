

# 💸 UPI Transactions Analysis Dashboard

This project provides an in-depth analysis of digital payment trends using Power BI. The focus of this dashboard is on **Advanced UI/UX Design**, **Dynamic Metric Switching**, and pixel-perfect formatting to create a highly interactive and professional user experience. 

## 🚀 Key Features & Techniques Demonstrated

* **Data Transformation (Power Query):** Cleaned and transformed raw transactional data from an Excel file, handling varying currencies (INR, USD, EUR, GBP) and transaction types.
* **Pixel-Perfect UI (Size & Position):** Applied mathematical alignment and specific X/Y coordinate positioning to organize 10 comprehensive slicers (Bank Name, City, Device Type, Gender, etc.) flawlessly across the canvas.
* **Dynamic Metric Switching:** Utilized **Bookmarks and Buttons** to allow users to toggle seamlessly between different visual types and metrics without cluttering the dashboard:
    * *Line Chart: Amounts*
    * *Column Chart: Amounts*
    * *Line Chart: Balances*
    * *Column Chart: Balances*
* **Synced Slicers:** Implemented "Sync Slicers" functionality across multiple report pages (Page 1 & Page 2) to ensure a continuous and seamless filtering experience for the end-user.

## 📂 Project Files
This repository folder contains the following core files:
1. **`UPI_Transactions_Analysis.pbix`**: The fully developed Power BI report.
2. **`Raw_Data.xlsx`**: The original dataset used for the analysis.
3. **`UPI_Transactions_Analysis.pdf`**: A static PDF export of the dashboard for quick viewing.

## 📊 Dashboard Overview
The dashboard tracks key financial metrics across 2024, answering critical business questions such as:
* How do transaction amounts and remaining balances trend month-over-month?
* Which cities (Bangalore, Delhi, Hyderabad, Mumbai) generate the highest transaction volumes?
* What is the distribution of transactions across different banks, device types, and age groups?

## 📝 How to Use This Project
1. Download the `.pbix` file.
2. Open it in **Power BI Desktop**.
3. Use the top navigation slicers to filter data by Bank, City, or Transaction Type.
4. Click the **Metric Switching Buttons** (e.g., *Line Chart Amounts*, *Column Chart Balance*) to test the interactive bookmarking features.
