## 🛠️ Technical Stack & Tools
* **Database:** Microsoft SQL Server (SSMS)
* **ETL/Transformation:** Power Query (M Language)
* **Analytics Engine:** Power BI Desktop (DAX)
* **Cloud Distribution:** Power BI Service (Fabric)
* **Connectivity:** On-premises Data Gateway (Personal Mode)
* **AI:** Azure Cognitive Services / AI Insights

## 🔐 Security & Architecture
Most projects in this repo include a robust security layer:
1. **Row-Level Security (RLS):** Dynamic and Static roles to ensure data privacy.
2. **Cloud Integration:** Secure data publishing and automated refresh cycles.
3. **Data Integrity:** Validation of SQL imports vs. Power BI datasets.
